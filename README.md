# llf_users_simple

### 移动端项目

## 安装依赖
```
npm install
```

## 运行项目
```
npm run serve
```

## 编译项目
```
npm run build
```

## 可能在项目中需要用到的插件安装及命令

```
npm install vant@bata  --save-dev
npm install lib-flexible  --save-dev
npm install vue-awesome-swiper --save
npm install better-scroll --save

```
